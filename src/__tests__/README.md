# Тесты

Этот проект использует Jest и React Testing Library для тестирования.

## Запуск тестов

```bash
# Запустить все тесты
npm test

# Запустить тесты в watch режиме
npm run test:watch

# Запустить тесты с покрытием кода
npm run test:coverage
```

## Структура тестов

- `src/utils/__tests__/` - тесты для утилитных функций
- `src/store/slices/__tests__/` - тесты для Redux слайсов
- `src/app/components/__tests__/` - тесты для React компонентов
- `src/app/workouts/[id]/__tests__/` - тесты для страниц

## Написанные тесты

### 1. Утилиты

- `normalizeVideoUrl.test.ts` - тесты для нормализации YouTube URL

### 2. Redux слайсы

- `uiSlice.test.ts` - тесты для UI состояния (модальные окна, загрузка, ошибки)
- `coursesSlice.test.ts` - тесты для управления курсами

### 3. Компоненты

- `deleteConfirmModal.test.tsx` - тесты для модального окна подтверждения удаления

## Добавление новых тестов

При создании нового компонента или функции создавайте соответствующий тестовый файл:

- Для компонентов: `ComponentName.test.tsx`
- Для утилит: `utilityName.test.ts`
- Для Redux: `sliceName.test.ts`

## Покрытие кода

Цель - достичь минимум 80% покрытия кода тестами для критически важных частей приложения.

# Sky_fitnessPro

[![Deploy to GitHub Pages](https://github.com/Veritas-lab/Sky_fitnessPro/actions/workflows/deploy.yml/badge.svg)](https://github.com/Veritas-lab/Sky_fitnessPro/actions/workflows/deploy.yml)

Сайт для фитнеса, который позволяет выбирать курсы и отслеживать прогресс тренировок.

## 🚀 Деплой

Сайт автоматически деплоится на GitHub Pages при каждом push в ветку `graduation-project`.

🌐 **Live сайт:** [https://veritas-lab.github.io/Sky_fitnessPro/](https://veritas-lab.github.io/Sky_fitnessPro/)

## 🛠️ Технологии

- Next.js 15
- React 19
- TypeScript
- Redux Toolkit
- CSS Modules
